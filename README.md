# HHU-Software-Engineering
# 河海大学计信院软件工程作业

# 操作说明⚠️
 - 用自己的账号登陆Github
 - fork按钮拷贝https://github.com/junyubb/HHU-Software-Engineering到你的账户下
 - 在相应的目录下添加自己的作业文件，文件名为自己的学号，如在第一次作业中添加1706010512.cpp
 - 用pull request提交作业
 - 等待管理员同意
